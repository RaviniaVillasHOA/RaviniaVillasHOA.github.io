Submitting Work Orders

Submitting a Work Order For Your Own Property

You will need to access Access Vantaca Portal

Here are Quick Instructions to Access or Create your Vanataca Account if you need them.

On the Vantaca Portal, find the Requests menu item on left side bar.

Select the General Request for ALL work orders (maintenance, landscaping, etc) except for Architectural Review Request.

Only select ARC Request if you need to submit an Architectural Review Request. Refer to the Architectural Review Committee documents concerning Architectural Review processes.

Submitting a Work Order For Common Property, another homeowner's property, or multiple properties.

Send an e-mail to WorkOrder@RaviniaVillasHOA.org

In the E-mail, besure to include your name and e-mail address as the submitter. Include a description of the workorder. Be as specfic as you can concering the location of the item in question. It's best if you can attached photo(s). If you are submitting a work order on behalf of another resident, please include their name (if known) and their property address. You should include enough information so that the receiptent of the work order can understand both the problem as well as the location.

Note: Never submit an ARC Request on behalf of another resident. You may be inadvertantly charged an ARC Requtest Fee.
