# Ravinia Villas HOA Website

This repository contains the source code and files for the official Ravinia Villas HOA website.
The site is hosted via GitHub Pages at https://RaviniaVillasHOA.org and https://www.RaviniaVillasHOA.org

## Purpose
- Provide HOA information to residents

## Repository Structure
- index.html        → Home page
- gate-access.html  → Various methods of entry gate and pedistran gate access
- new-resident.html - Information for New Residents
- work-orders.html  → How to submitting Work Orders page
- contact.html      → Contact information
- assets/           → Images, PDFs, and other media
  - style.css          → Stylesheets
  - community.jpg      - Community Front Gate Image

## Contributing
- All changes should be made via pull requests or directly by authorized organization members.
- Branch protection is enabled on `main` to prevent accidental changes.
- Board members with Owner or Maintainer roles can update content.
